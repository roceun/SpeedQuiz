## 개발용 추천 메모

- 각 카테고리에서 게임 시작 시 10개를 중복 없이 랜덤 추출
- 단어 길이와 난이도가 한 번에 너무 몰리지 않도록 이후 난이도 태그 추가 가능
- 첫 버전에서는 카테고리당 30개 이상 확보한 현재 목록으로도 충분히 시작 가능

## 다음 정리 후보

- 카테고리별 쉬움/보통 난이도 구분
- 설명이 애매할 수 있는 단어 제거
- 실제 플레이 테스트 후 자주 패스되는 단어 교체
